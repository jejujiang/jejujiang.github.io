[Hello 算法](https://www.hello-algo.com)  
数据结构无处不在：大到社会网络，小到地铁线路，许多系统都可以建模为”图”；大到一个国家，小到一个家庭，社会的主要组织形式呈现出”树”的特征；冬天的衣服就像”桟”，最先穿上去的最后才能脱下；羽毛球桶则如同”队列”，一端放入，另一端取出；字典就像一个”哈希表”，能够快速查找目标词条
```ruby
# for 循环
def for_loop(n)
  res = 0

  for i in 1..n
    res += i
  end

  return res
end

# while 循环
def while_loop(n)
  res = 0
  i = 1 # 初始化条件变量

  while i <= n
    res += i
    i += 1 # 更新条件变量
  end

  return res
end
```
